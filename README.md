1.5 Зв'язність та цикли
Чи є граф зв'язним? → ТАК
З будь-якої вершини можна дістатися до будь-якої іншої, наприклад: 3 → 1 → 0 → 2 → 6. Усі 7 вершин входять до однієї компоненти зв'язності.
Чи містить граф цикли? → НІ
КритерійЗначенняКількість вершин (n)7Кількість ребер (e)6Виконується e = n − 1?6 = 7 − 1 ✓
Граф є зв'язним і має рівно n − 1 = 6 ребер → це дерево. У дереві між будь-якими двома вершинами існує рівно один шлях, тому циклів немає. Додатково: вершини 3, 4, 5, 6 мають ступінь 1 (листя) — з них цикл побудувати неможливо.
Висновок: граф є зв'язним деревом без циклів.
